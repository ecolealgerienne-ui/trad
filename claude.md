# Projet trad - École Algérienne UI

## Description du projet

Ce projet contient les fichiers de traduction pour l'interface utilisateur de l'École Algérienne (ecolealgerienne-ui).

## Structure du projet

Ce dépôt est organisé pour gérer les traductions multilingues de l'application École Algérienne.

## Technologies

- Fichiers de traduction (JSON/i18n)
- Gestion de traductions multilingues

## Langues supportées

Les langues suivantes sont ou seront supportées :
- Français (fr)
- Arabe (ar)
- Anglais (en)

## Organisation des fichiers

Les fichiers de traduction sont organisés par langue et par module de l'application.

## Contribution

Pour ajouter ou modifier des traductions :
1. Créer une branche de fonctionnalité
2. Modifier les fichiers de traduction appropriés
3. Soumettre une pull request

## Notes pour Claude

- Ce projet fait partie de l'écosystème ecolealgerienne-ui
